## Pipeline Overview

8 phases, ~104 template blocks, organized by coaching lens/skill family:
1. Exemplar authoring (voice calibration)
2. Skill-family batch calls (4 lens groups)
3. Cross-cutting blocks (composite skills, environmental)
4. DDx-specific blocks (blocker explanations, strategy derivations)
5. Human review pass
6. JSON compilation
7. Integration testing
8. Iteration
